# College Hostel Management System

**What we did with HTML, CSS, and JS:**
We fully designed and styled all the visual screens (like the student dashboard, login pages, and leave forms) from scratch, writing our own CSS to make it look incredibly professional.

**What we did with React:**
We successfully converted all of those separate HTML pages into a modern React application. We mapped everything into one lightning-fast web app that lets you click through all the different hostel features instantly without the browser ever having to reload!
