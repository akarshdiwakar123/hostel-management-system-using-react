# College Hostel Counselling System (React)

A professional, enterprise-grade Hostel Management System web application converted into a blazing fast Single Page Application (SPA) using React and Vite. 

This digital initiative facilitates Faculty, Staff, and Students to easily access and manage hostel services on one common platform.

## 🚀 Features

### 👨‍🎓 Student Portal
* **Dashboard Overview**: Get a bird's-eye view of your hostel status including Room Number, Fee Status, and Active Leaves/Complaints.
* **View Profile**: Seamlessly view student registry information.
* **Room Details**: Check wing, block, room type, and roommate information.
* **Fee Details**: Keep track of paid and pending semester hostel and mess fees.
* **Mess Menu**: View the day-by-day weekly mess schedule.
* **Apply Leave**: Apply for out-station absences and track approval statuses.
* **Register Complaint**: Log maintenance issues directly with the hostel staff (Electrical, Plumbing, Cleaning, etc.).

### 👔 Employee Portal
* **Administration Dashboard**: High-level Analytics on capacity, pending complaints, leave requests, and fee deficits.
* **Manage Students**: View student allocations and manage room placements.
* **Complaints Inbox**: Track, manage, and resolve student-raised maintenance tickets.
* **Leave Approvals**: View submitted leave requests and grant or deny them.
* **Fee Records**: Verify offline payment receipts and track deficit accounts.

## 🛠️ Technology Stack
* **Frontend Library:** React
* **Build Tool:** Vite
* **Styling:** Vanilla CSS (Professional enterprise design logic)
* **Architecture:** State-driven Single Component View Controller (No external routing dependencies)

## 📁 Project Structure

To keep the application elegantly simple and fast, the entire system functionality is mapped perfectly inside a single-component architecture:
* `src/App.jsx`: Houses the State-Controller and handles structural conditional rendering for all module views seamlessly.
* `src/index.css`: Contains the robust, global UI styling.
* `src/main.jsx`: The React DOM entry point.

## 💻 Getting Started Locally

1. **Navigate into the project directory**:
   ```bash
   cd hostel-management-system-using-react/my-app
   ```

2. **Install the dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **View the Application**: Open the terminal's provided `localhost` link in your browser to experiment with the portals!
